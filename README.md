# SE302 Homework 02 – Sweet Shop (Playwright)

This repository contains the solution for **SE302 – Software Testing and Maintenance, Homework 02**.  
The assignment focuses on **manual test case design** and **automated web testing** using **Playwright** with the **Page Object Model (POM)** structure.

---

##  Project Overview

- **Website under test:** https://sweetshop.netlify.app/
- **Testing type:** Manual testing + Automated end-to-end testing
- **Automation tool:** Playwright
- **Language:** TypeScript
- **Design pattern:** Page Object Model (POM)

From 10 manually written test cases, **5 were selected and automated** as required by the assignment.

---

## Automated Test Cases

The following manual test cases were automated:

| Test Case ID | Description |
|-------------|------------|
| TC01 | Homepage loads successfully |
| TC03 | Navigation to About and Login works |
| TC05 | Add to basket updates basket indicator |
| TC08 | Login blocks empty submission (negative test) |
| TC10 | Script-like input does not trigger alert |

The remaining test cases were executed manually and documented in the PDF report.
